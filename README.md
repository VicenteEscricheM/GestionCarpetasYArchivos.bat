# GestionCarpetasYArchivos.bat
--ESP--
Archivo por lotes .bat que proporciona entorno gráfico para la creación,
modificación y eliminación avanzada de carpetas y archivos a través de consola.
Creado en 2024 como práctica independiente en el curso IFCT0209 Sistemas Microinformáticos.

* Es recomendable ejecutar el programa como administrador para su correcto funcionamiento.
* Este script se programó en el contexto de unas prácticas independientes dentro de un curso de Sistemas Microinformáticos, algunas
  funcionalidades no se encuentran acabadas o plenamente operativas.
* ¡Disfrútalo con cuidado!, algunas funcionalidades pueden alterar el sistema de forma irreversible. Por si acaso, recuerda crear un punto de restauración ;)


--ENG--
Batch file created for advanced folder and files management, allowing the user to create, modify and remove files in a graphic enviroment in cmd.
Created in 2024 as an independent project within the context of an elementary computer science course.

* Its highly recomended to run this program as admin.
* This script was programmed within the context of and independent project. Therefor, you may find that some tools or functionalities are unfinished or
  not completely functional.
* Be careful while running this program! some functionalities may result in irreversible changes. Just in case, I'd highly recommend the creation
  of a restoration point.
